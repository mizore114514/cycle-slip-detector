# 周跳探测小程序 — 深色科技风前端重设计

**日期**: 2026-05-16 | **状态**: 待实现

## 1. 概述

将 Streamlit 应用从默认浅色主题改造为深色科技风，提升专业感和视觉舒适度。改动范围：主题配置、CSS 注入、Plotly 图表重设计、首页/侧边栏/结果页布局优化。不改变核心算法和交互逻辑。

## 2. 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#0D1117` | 极深蓝黑 |
| 卡片/面板 | `#161B22` | 微亮深灰 |
| 边框 | `#30363D` | 暗色分割线 |
| 主强调色 | `#00D4AA` | 霓虹青（标题、高亮） |
| 次强调色 | `#58A6FF` | 科技蓝（链接） |
| 警告色 | `#F78166` | 暖橙（周跳标注） |
| 字体主色 | `#E6EDF3` | 近白灰 |
| 字体次色 | `#8B949E` | 中灰辅助 |

## 3. Streamlit 主题配置

修改 `.streamlit/config.toml`，增加 `[theme]` 节，设置 primaryColor/secondaryColor/backgroundColor 等。

## 4. 首页重设计

从纯文字列表改为 Hero 式布局：

- 大号图标 + 中英文主副标题
- 三步操作流程卡片（上传 → 调参 → 查看）
- 底部三列能力卡片（卫星系统 / 文件格式 / 功能特性）
- 背景叠加微弱网格纹理（CSS 实现）

## 5. 侧边栏

- 顶部标题区：应用名 + 副标题，霓虹青强调
- 分区用细分割线隔开：数据输入 / 检测参数 / 关于
- 上传区：虚线边框框选区域
- 参数区：每个滑块旁显示当前值，圆点标记当前位置
- 关于区：小卡片展示 GF/MW 公式

## 6. 结果表格

- 表头霓虹青背景
- 行间斑马纹（深灰交替）
- 周跳数值超过阈值标记为橙色
- CSV 导出按钮用霓虹青强调

## 7. 可视化图表

图表全面应用暗色主题风格：

### GF 时序图
- 暗色画布背景 `#161B22`
- GF 曲线：科技蓝 `#58A6FF`，线宽 2
- 周跳标注：暖橙 `#F78166` × 标记，size 12
- 网格线细暗 `#30363D`
- 字体白色，title 霓虹青

### MW 时序图
- 暗色画布背景
- MW 原始值：霓虹青 `#00D4AA`，线宽 1.5
- 滑动均值：科技蓝 `#58A6FF`，虚线
- 阈值上下界：暖橙 `#F78166`，细虚线
- 周跳标注：暖橙 × 标记
- hover 提示框暗色样式

## 8. 自定义 CSS

通过 `st.markdown(unsafe_allow_html=True)` 注入 CSS：
- 全局字体调整为 system-ui 无衬线
- 卡片式容器（圆角 8px，背景色，阴影）
- 按钮霓虹青描边
- 滑块轨道改为主强调色
- 分割线颜色统一
- 首页网格纹理背景

## 9. 受影响的文件

| 文件 | 改动 |
|------|------|
| `.streamlit/config.toml` | 新增 `[theme]` 节 |
| `app.py` | 首页重写、CSS 注入、暗色背景设置 |
| `ui/sidebar.py` | 分区样式、数值显示、公式卡片 |
| `ui/results.py` | 表格样式、按钮强调色 |
| `ui/charts.py` | Plotly 暗色模板、配色、hover 样式 |

## 10. 不做的事

- 不改变算法逻辑
- 不增加新依赖
- 不动 Streamlit 核心交互流（上传 → 检测 → 展示）
- 不用自定义 Streamlit 组件（保持零前端依赖）
