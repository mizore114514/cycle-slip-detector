# 周跳探测小程序 — 使用说明

## 一、运行方式

### 方式一：在线访问（推荐，无需安装）

直接浏览器打开：**https://cycle-slip-detector-ghjwgzunmpch2gyqpfacpt.streamlit.app**

### 方式二：双击启动

1. 在 Finder 中找到 `启动.command`
2. **双击**即可启动，Terminal 会自动打开，浏览器也会自动跳转到程序界面
3. 如浏览器未自动打开，手动访问 **http://localhost:8501**

### 方式三：源码运行（需提前安装 Python）

1. 打开终端，进入项目目录：
   ```
   cd 周跳探测小程序
   ```
2. 安装依赖（仅首次需要）：
   ```
   pip install -r requirements.txt
   ```
3. 启动程序（以下任选一种）：
   ```
   ./启动.command
   ```
   或
   ```
   python3 -m streamlit run app.py
   ```
4. 浏览器访问 **http://localhost:8501**

## 二、操作步骤

### 第1步：上传数据
- 在页面左侧「📁 数据输入」区域，点击 **Browse files** 按钮
- 选择 RINEX 双频观测文件（`.obs`、`.rnx` 等格式）

### 第2步：查看结果
上传后程序自动解析并检测周跳，页面主区域有两个标签页：

#### 📋 周跳结果
- 表格显示检测到的所有周跳，包含：
  - **卫星**：卫星编号（如 G01、G05）
  - **时间**：周跳发生的时刻
  - **GF跳变(米)**：Geometry-Free 组合的跳变量
  - **MW跳变(周)**：Melbourne-Wübbena 组合的跳变量
- 点击 **📥 导出 CSV** 按钮可下载结果报告

#### 📈 可视化分析
- 通过下拉框选择不同卫星
- 上方图表：GF 组合时序图，红色 ✕ 标记周跳位置
- 下方图表：MW 组合时序图，含滑动均值线和检测阈值线，红色 ✕ 标记周跳位置

### 第3步：调整参数（可选）
- **GF 检测阈值**：默认 0.05 米，值越小越敏感（更容易检出小周跳，但可能误检）
- **MW 滑动窗口**：默认 30 个历元，用于计算滑动均值
- **MW 检测阈值**：默认 1.0 周，值与滑动均值偏差超过此值即标记为周跳

## 三、算法简介

本程序采用 **TurboEdit** 方法：

| 组合 | 公式 | 特点 |
|------|------|------|
| **GF 组合** | λ₁·Φ₁ − λ₂·Φ₂（米） | 对小周跳敏感，消除几何项和钟差 |
| **MW 组合** | 宽巷相位 − 窄巷伪距（周） | 消除几何项、电离层和钟差，适合探测大周跳 |

两种组合互补，检测结果取并集，确保不漏检。

---

## 四、注意事项

- 仅支持 **双频** RINEX 观测文件（需含 L1/L2 + P1/P2 或等效观测量）
- 支持 RINEX 2.x 和 3.x 格式
- 支持 GPS / BDS / Galileo / GLONASS 卫星系统
- 如果数据量较大，解析可能需要几十秒，请耐心等待
